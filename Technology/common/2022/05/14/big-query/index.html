2. Commands hay dùng

Set active account service with file key

  gcloud auth activate-service-account --key-file config_path

Trong đó config_path là đường dẫn đến file config Và file config có nội dung ví dụ như sau:

  {
      "type": "service_account",
      "project_id": "my-project-dev",
      "private_key_id": "private_key_id",
      "private_key": "-----BEGIN PRIVATE KEY-----\\n....\\n-----END PRIVATE KEY-----\\n",
      "client_email": "michaelbell@my-project-dev.iam.gserviceaccount.com",
      "client_id": "client_id",
      "auth_uri": "https://accounts.google.com/o/oauth2/auth",
      "token_uri": "https://oauth2.googleapis.com/token",
      "auth_provider_x509_cert_url": "https://www.googleapis.com/oauth2/v1/certs",
      "client_x509_cert_url": "https://www.googleapis.com/robot/v1/metadata/x509/michaelbell%my-project-dev.iam.gserviceaccount.com"
  }

Dataset

Create new Dataset

  bq --location=us mk --dataset project_id:data_set

Ví dụ: Tạo dataset airbnb

  bq --location=us mk --dataset my-project-dev:airbnb

Remove dataset

  bq rm -r -f -d project_id:data_set

Trong đó:

  rm: remove
  -r: recursive
  -f: fork
  -d: 

  Ví dụ: bq rm -r -f -d my-project-dev:airbnb

Load file to table Biquery

Load from local file

  bq load --source_format=CSV --autodetect  data_set.table_id "file_path"

Load from GCS files

  bq load --source_format=NEWLINE_DELIMITED_JSON --autodetect data_set.table_id "gcs_file_path"

Ví dụ:

  bq load --source_format=NEWLINE_DELIMITED_JSON --autodetect data_set.table_id "gs://my-bucket-dev/development/user.json.gz"

Load file with schema defined

  bq load --source_format=NEWLINE_DELIMITED_JSON --ignore_unknown_values data_set.table_id "gcs_file_path" "schema_file_path"

VD:

  bq load --source_format=NEWLINE_DELIMITED_JSON --ignore_unknown_values data_set.table_id "gs://my-bucket-dev/development/user.json.gz" "~/Desktop/schema.json"

Chú ý:

  source_format=NEWLINE_DELIMITED_JSON để load files data Json
  source_format=CSV để load files data CSV
  --autodetect: BQ sẽ tự detect các fields, field type tương ứng với các data được load từ file lên BQ
  --ignore_unknown_values: Dựa vào thông tin trong file schema.json, BQ sẽ chỉ lấy những data trong file data theo những trường đã được khai báo trước.

Show Schema of table

  bq show --schema --format=prettyjson data_set.table_id > /tmp/schema_of_table

Remove table

  bq rm -t data_set.table_id

Run query

  bq query --nouse_legacy_sql \
  'query_statement'

Ví dụ:

  bq query --nouse_legacy_sql \
  'SELECT COUNT(*) FROM airbnb.users'
